## Design Thinking

Before coding, commit to a **bold aesthetic direction**:

- **Purpose**: What problem does this solve? Who uses it?
- **Tone**: Pick a clear direction — brutally minimal, maximalist, retro-futuristic, organic, luxury, playful, editorial, brutalist, art deco, soft/pastel, industrial. Commit fully.
- **Differentiation**: What makes this unforgettable?

## Aesthetics Rules

**Typography**: Choose distinctive, characterful fonts. Pair a display font with a refined body font. NEVER default to Inter, Roboto, Arial, Space Grotesk, or system fonts.

**Color & Theme**: Dominant colors with sharp accents. Use CSS variables. Draw from IDE themes, cultural aesthetics, or specific moods for inspiration. Avoid purple-gradient-on-white and other clichéd AI palettes.

**Motion**: Focus on high-impact moments — one well-orchestrated page load with staggered `animation-delay` beats scattered micro-interactions. CSS-only for HTML; Motion library for React. Surprise with scroll-triggers and hover states.

**Spatial Composition**: Asymmetry, overlap, diagonal flow, grid-breaking elements. Generous negative space OR controlled density — not the safe middle.

**Backgrounds & Texture**: Layer gradients, noise, geometric patterns, grain overlays, dramatic shadows. Create atmosphere and depth, never flat solid colors.

## Anti-Patterns (Hard Avoid)

- Generic font stacks (Inter, Roboto, Arial, system-ui)
- Converging on the same "safe" font across generations
- Purple/blue gradient cards on white backgrounds
- Predictable symmetric layouts with uniform spacing
- Cookie-cutter component patterns with no contextual character
- Timid, evenly-distributed color palettes

## Execution

Match complexity to the vision. Maximalist designs need elaborate animations and layered effects. Minimalist designs need surgical precision in spacing, type, and subtle details. Every design should feel like it was made _for this specific context_ — never interchangeable.

Vary between light/dark themes, font choices, and aesthetic directions across generations. Never repeat yourself.
